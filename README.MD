# 🏢 Zelatech - Gestão de Manutenção de Condomínios

O **Zelatech** é uma plataforma moderna e responsiva voltada para a gestão e controle de manutenções em condomínios. O sistema foi projetado para integrar e facilitar a comunicação entre Síndicos, Zeladores (Equipe de Manutenção) e Moradores, oferecendo um controle completo de ponta a ponta.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando uma stack moderna e de alta performance:
* **Next.js & React.js**: Framework de interface e renderização.
* **TypeScript**: Tipagem estática para maior segurança do código.
* **Tailwind CSS & Shadcn UI**: Estilização ágil e componentes de interface consistentes (Data Tables, Modais, Cards e Formulários).
* **PostgreSQL**: Banco de dados relacional robusto.
* **Prisma ORM**: Modelagem de dados e interações com o banco.

## ✨ Principais Funcionalidades

O projeto possui uma arquitetura modular focada nas necessidades de cada usuário:

### 📊 Dashboards Personalizados
* **Síndico**: Visão gerencial com métricas de custos, chamados abertos e manutenções preventivas atrasadas.
* **Zelador / Manutenção**: Visão operacional com lista de tarefas diárias, chamados urgentes e alertas de material.
* **Morador**: Acompanhamento exclusivo de seus próprios chamados e visualização do mural de avisos.

### 🛠 Gestão de Chamados e Ordens de Serviço (OS)
* **Abertura de Chamado**: Formulários detalhados com descrição, prioridade e upload de fotos.
* **Detalhes da OS**: Acompanhamento de histórico, materiais utilizados, tempo gasto e alteração de status (Aberto, Em Andamento, Concluído).

### 📅 Manutenções Preventivas
* Calendário para agendamento e acompanhamento de rotinas de manutenção.
* Checklists de execução interativos para controle passo a passo pelo zelador.

### 📦 Estoque e Fornecedores
* Inventário de materiais com indicadores de quantidade e alertas de reposição.
* Diretório de fornecedores para acesso rápido a contratos e contatos de terceirizados.

### 📈 Comunicação e Relatórios
* Mural de avisos para comunicados do síndico.
* Relatórios visuais de custos e desempenho operacional.
